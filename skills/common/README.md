# Common Skills

这里存放可迁移到其他项目的通用版 skill。

## 包含内容

- `project-release-qa`：适合大多数软件项目的审查、修复、测试、构建、打包流程。
- `project-prd-gap-implementation`：适合所有有 PRD 或需求文档的项目。
- `browser-bridge-troubleshooter`：适合所有使用浏览器桥接、浏览器插件或网页自动化的项目。

## 迁移方式

复制目标 skill 目录到其他项目或 Codex skills 目录即可。迁移后第一次使用时，应先让 Codex 读取目标项目结构，自动识别测试命令、构建命令和打包命令。
